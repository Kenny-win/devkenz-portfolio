## 📦 Getting Started

To run this project locally on your machine, follow these simple steps:

1.  **Clone the repository:**
    ```bash
    git clone [https://github.com/Kenny-win/devkenz-portfolio](https://github.com/Kenny-win/devkenz-portfolio)
    cd your-repo-name/frontend
    ```

2.  **Install dependencies:**
    ```bash
    npm install
    ```

3.  **Run the development server:**
    ```bash
    npm run dev
    ```
    Open [http://localhost:5173](http://localhost:5173) in your browser to see the result.

---

## 📸 Profile Picture Setup

To replace the profile placeholder with your actual photo:
1. Prepare a square ($1:1$ ratio) photo of yourself.
2. Save it inside the `frontend/public/` directory.
3. Rename the file to `profile.jpg`.
4. The application will automatically detect and render your image!
